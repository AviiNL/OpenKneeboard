# Game Compatibility

| Graphics API | Non-VR | OpenXR | Oculus | SteamVR |
| -------------|--------|--------|--------|---------|
| OpenGL       | ❌ | ❌ | ❌ | ✅ |
| Direct3D 11  | ✅ | ✅ | ✅ | ✅ |
| Direct3D 12  | ❌ | ✅ | ✅ | ✅ |
| Vulkan       | ❌ | 🧪 | ❌ | ✅ |

🧪: OpenKneeboard's support for Vulkan+OpenXR is currently experimental.
